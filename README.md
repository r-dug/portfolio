## 🚀 Projects

### [TutorAIL](https://tutorail.app)
Agentic tutoring platform — real-time voice pipeline, 20 agent tools (code sandbox, RAG, drawing eval, web search), 4 pluggable memory strategies for context management, and a distillation pipeline for collecting training data. Includes an eval framework (4 deterministic + 1 LLM grader) and a full course publishing model. ~40K LOC, 212 passing tests, 79 API routes. Deployed via systemd + nginx + certbot.

### [bounty-os](https://github.com/r-dug)
Multi-agent bug bounty orchestrator — 10 LLM agents in a sequential pipeline (recon → scanning → exploitation → reporting → HackerOne submission) with adaptive feedback loops, confidence scoring, and an AI pentest agent implementing 5 academic attack techniques. 17K LOC Python, 193 targets scouted, zero scope violations.

### [Neonatal AI](https://github.com/Neonatal-AI)
LLM-powered clinical documentation tool built with medical professionals at UofL. Handles the paperwork so neonatal care workers can focus on patients.

### [Federated Learning Research](https://github.com/r-dug)
Research into federated learning aggregation methods. Proposed a novel approach using unsupervised clustering on gradient properties. Ongoing work at UofL.

### [Style Mimicry Robustness](https://github.com/r-dug) *(in progress)*
AI security research testing whether perturbation-based artist protections hold up against countermeasures. 4×4 experimental matrix across 639 images, 26 artists, and 6 art-historical styles on Stable Diffusion 3.5 Large. Automated LoRA fine-tuning pipeline with ~40–50 GPU-hours invested so far. Benchmarking and human evaluation phases are next.

### [Fungal Species Classifier](https://github.com/r-dug/GCViT_Classifier)
1,000-species fungal classification using a custom GCViT (Global Context Vision Transformer) built from scratch in TF/Keras. 194K iNaturalist images, DBSCAN outlier filtering (~20K removed), staged unfreezing, TFLite conversion (433 MB → 32 MB), and a Kotlin/Android app with CameraX real-time inference and species metadata.

### [Garden Wall](https://github.com/r-dug/garden_wall)
Raspberry Pi + object detection to catch garden pests and fire a solenoid valve to scare them off. Chipmunks hate it.

### [Notes GPT](https://github.com/r-dug/notes_GPT)
CLI tool that turns lecture videos into HTML study notes using LLMs. Built it for myself, works well enough to share.
